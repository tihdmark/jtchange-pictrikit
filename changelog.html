Changelog

Stay up to date with the latest features and improvements.

v2.2.0 January 2026

Original Scale Mode for Linear Layout

  • New Added "Original" scale mode for Linear layouts
  • New Alignment options (Top/Center/Bottom) for original scale mode
  • Improved Two distinct modes: Uniform (visual consistency) vs Original (true proportions)
  • Improved Better support for mixing desktop and mobile screenshots
v2.1.0 January 2026

Linear Export Enhancement

  • Improved Linear layout export now aligns images by max height/width
  • Improved Large images keep original resolution, small images scale up
  • Improved High-quality image smoothing for scaled images
  • Fixed Export quality loss when mixing different sized images
v2.0.0 January 2026

Major Release: Complete Redesign

  • New 5 layout modes: Linear, Focus, Grid, Compare, Frames
  • New Device frame mockups (Browser, Phone, Tablet)
  • New Background color picker with presets
  • Improved Completely redesigned user interface
  • Improved Better export quality and performance
v1.5.0 December 2025

Style Controls Update

  • New Adjustable padding and gap controls
  • New Corner radius customization
  • Improved Real-time preview updates
  • Fixed Image quality on high-DPI displays
v1.0.0 November 2025

Initial Release

  • New Basic screenshot layout tool
  • New Drag and drop image upload
  • New PNG export functionality
  • New Copy to clipboard feature
